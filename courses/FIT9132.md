# FIT9132 选课指南｜Introduction to databases｜数据库导论

## 一句话总结

FIT9132 是适合零基础的数据库入门必修课。核心是理解关系型数据库的基本结构，掌握 ER 建模、SQL 查询，以及如何设计和管理一个基础数据库。

## 课程内容

主要涉及：

- SQL
- ER 建模
- 数据库设计
- 数据表
- 主键 / 外键
- 表关系
- 基础数据库管理

---

## Assessment 类型

常见考核形式包括：

- Quiz / Test 1：15%
- Quiz / Test 2：15%
- Artefact 1 - Database Design：35%
- Artefact 2 - Creating, Populating and Manipulating Databases：35%

具体以当学期 Handbook / Moodle 为准。

---

## 选课建议

建议重点练 SQL 和 ER 建模，尤其是：

- 主键
- 外键
- 表关系
- Database schema
- Normalisation
- 数据库设计规范

这门课整体逻辑比较清楚，但 workload 不低。很多分数来自实际设计和操作，建议不要把作业拖到最后一周再开始。

---

## 学姐提醒

FIT9132 对零基础同学相对友好，但不代表可以裸学。前几周一定要把 SQL 基础打牢，后面做 Artefact 的时候会轻松很多。

如果你未来想走 Data / BI / Backend / Software 方向，这门课属于比较值得认真学的基础课。

## 详细选课搭配以及求职方向

FIT9132 本身是一门数据库基础课，但它和后续选课、求职方向关系比较大。

如果你后续想走：

- Data Analyst / BI
- Backend Development
- Software Development
- Data Engineering
- Business Information Systems

就需要提前想清楚这门课后面怎么搭配 Python、数据分析、软件开发或项目课。
更具体的选课搭配、避坑组合和不同求职方向推荐因人而异。

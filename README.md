## 一、资料库定位

这是一个面向 Monash IT / CS / AI / DS 学生的课程排雷资料库。

资料库会整理课程分析表中的部分内容，帮助同学在选课前快速了解：

- 这门课大概学什么
- 是否适合零基础
- 是否有考试
- 是否需要组队
- 作业压力大不大
- 主要风险点在哪里
- 适合什么求职方向
- 选课前需要注意什么

本资料库基于 Monash Handbook、课程结构、公开信息和往届学生反馈整理，主要帮助 IT / CS / AI / DS 学生提前了解课程风险，减少盲选和踩坑。

---

## 二、适合谁看

这个资料库适合：

- 准备入学 Monash IT / CS / AI / DS 的新生
- 正在纠结选课、stream、elective 的同学
- 想提前了解课程难度和考核方式的同学
- 担心选错课、卡前置课、影响毕业进度的同学
- 想按照求职方向规划课程组合的同学

如果你现在的问题是：

- “这门课到底难不难？”
- “这门课有没有考试？”
- “这门课适不适合零基础？”
- “我想走 Data / AI / Software / Cybersecurity，该怎么选？”
- “哪些课容易组队踩坑？”
- “哪些课作业压力比较大？”

那这个资料库会比较适合你。

## 四、公开版说明

目前本资料库为公开版，只开放部分课程样例。

公开版会展示：

- 部分课程详细分析
- 课程代码和课程基础分类
- 新生常用工具和 App
- 选课前需要注意的风险点
- 不同求职方向的选课思路

其余课程会先展示课程代码、课程名称和基础分类，后续会逐步更新。

完整版课程分析表会包含更完整的内容，例如：

- 课程难度
- 考核方式
- 是否有 final
- 是否需要组队

- 学姐建议
- 往届学生反馈

- 选课组合建议
